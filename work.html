Weather Project

I created a web-based weather app using HTML, CSS and JavaScript React. Integrated with the SheCodes API, it provides real-time weather data for cities worldwide. Users can enter any city name to obtain an accurate forecast, including temperature, humidity, wind speed, and weather conditions. The app also includes a Fahrenheit converter and a basic database that remembers the user's preferred temperature display and last searched city.
